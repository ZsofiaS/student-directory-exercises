# Student Directory Exercises
The student directory scripts that allow you to manage the list of students enrolled at Villains Academy.</br>
New repo created from Student Directory for additional exercises.

`directory_step14.rb` is an amended and refactored version, with additional functionality.</br>
`prog_reads_itself.rb` is a little script that displays its own source code.

## How to use ##

```shell
ruby directory.rb
```
